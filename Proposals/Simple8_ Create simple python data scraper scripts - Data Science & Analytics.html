Data Scraper Specification

1. OVERVIEW, Requirements-
Create a python controller which creates, collects, and closes
    circuits on tor, an internet overlay connection creation network.
Convert server ID(hash) to server IP through a lookup.
Return a table of circuits: date/time and 3 IPS addresses on each line.

a. Close all existing circuits.
b. Create a new circuit. Tell tor "extendcircuit SP 0"
c. Convert/store node ID(hash) to IP using the tor control procotol (below).
c. Store timestamp and IP of each of 3 nodes in the circuit
e. Loop- repeat at Step 1.a.

Specification:

Use tor’s control port. All commands are available are in the tor control spec at https://gitweb.torproject.org/torspec.git/tree/control-spec.txt. Tor’s STEM library, https://stem.torproject.org/index.html, may possibly be useful.

1. Connect to the tor control port (generally tcp localhost 9051 or 9150)
2. "authenticate" as documented here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n352
3. "getinfo circuit-status" for a list of current active circuits
  as documented here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n611
4. Close existing circuits, using the CloseCircuit command, documented here,
   https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n1032
5. Create a new circuit, as seen here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n895

6. "getinfo version desc/name/relayname" replace relayname with the one you want to lookup.
  as documented here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n888
  tor returns the full descriptor which includes the router line:
  for example getinfo version desc/name/hypnos returns the desc"router hypnos 217.16.182.20 9001 0 9030"

IF there is no relay name and only fingerprint:

  "getinfo dir/server/fp/<fingerprint>" should return the descriptor by the fingerprint
  as documented here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n742

7. parse out the IP address and run getinfo ip-to-country/ipaddress to return the country
  as documented here, https://gitweb.torproject.org/torspec.git/tree/control-spec.txt#n721
  for example, getinfo ip-to-country/217.16.182.20 returns 250-ip-to-country/217.16.182.20=cz
